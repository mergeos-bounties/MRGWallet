# Chrome Extension: Load Unpacked Walkthrough

This guide walks you through loading the MRGWallet Chrome extension as an **unpacked extension** in Developer mode.

---

## Prerequisites

- [Google Chrome](https://www.google.com/chrome/) (v88+, MV3 compatible)
- MRGWallet extension build output (`dist/extensions/chrome/`)

## Step 1: Build the extension

```bash
git clone https://github.com/mergeos-bounties/MRGWallet.git
cd MRGWallet
npm install
npm run build:extensions
```

## Step 2: Open Chrome Extensions page

1. Open Chrome and navigate to: `chrome://extensions/`
2. Or click: **Menu (в‹®) в†’ Extensions в†’ Manage Extensions**

## Step 3: Enable Developer mode

Toggle **Developer mode** ON (top-right corner).

![Developer mode toggle](screenshots/chrome-dev-mode.png)

## Step 4: Load unpacked extension

1. Click **Load unpacked** button (top-left)
2. Select the `dist/extensions/chrome/` directory
3. Click **Select Folder** (or **Open**)

![Load unpacked button](screenshots/chrome-load-unpacked.png)

## Step 5: Verify the extension

After loading, you should see **MRGWallet** in the extensions list:

- **Status**: ON (toggle enabled)
- **Version**: matches `package.json`
- **ID**: auto-generated by Chrome

![Extension loaded](screenshots/chrome-extension-loaded.png)

## Step 6: Pin to toolbar

1. Click the **Extensions** puzzle icon (top-right toolbar)
2. Find **MRGWallet**
3. Click the **pin** icon to keep it visible

![Pin extension](screenshots/chrome-pin-extension.png)

## Step 7: Open the wallet

Click the MRGWallet icon in the toolbar to open the popup.

![Wallet popup](screenshots/chrome-wallet-popup.png)

## Troubleshooting

| Symptom | Fix |
|---------|-----|
| "Manifest is not valid" | Run `npm run build:extensions` again to regenerate |
| Icons not showing | Ensure `dist/extensions/chrome/icons/` contains `icon16.png`, `icon48.png`, `icon128.png` |
| Popup blank | Check browser console (F12) for JS errors |
| Extension not found after reload | Go to `chrome://extensions/` and verify it's enabled |

## Reload after changes

After modifying the source code and rebuilding:

1. Go to `chrome://extensions/`
2. Find **MRGWallet**
3. Click the **в†» (Reload)** icon
4. Or press `Ctrl+R` / `Cmd+R` with the extension card focused
